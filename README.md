## Run Locally

Clone the project

```bash
  git clone https://github.com/volcar144/arcium-os.git
```

Go to the project directory

```bash
  cd arcium-os/arciumos
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm start
```
